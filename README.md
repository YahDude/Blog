## Quick Start

Fisrt, publish `hexo_img_relocate` to `/tmp`:

```shell
blog$ cd hexo-img-relocate
hexo-img-relocate$ npm run tarball
```

Then, install the dependencies:

```shell
blog$ npm install
```

Finally, run local server and writing:

```shell
blog$ npm run server
```
